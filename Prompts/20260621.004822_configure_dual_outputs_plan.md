Modify the GitHub Actions workflow file `.github/workflows/build.yml` to trigger two separate compilation targets:
1. **Framework-Dependent Build**: Compiles the app by overriding the properties `-p:SelfContained=false` and `-p:WindowsAppSDKSelfContained=false`, exporting the output to `build-output/framework-dependent`.
2. **Self-Contained Build**: Compiles the app by overriding the properties `-p:SelfContained=true` and `-p:WindowsAppSDKSelfContained=true`, exporting the output to `build-output/self-contained`.

Upload both builds as distinct zip artifacts (`StickerApp-Framework-Dependent` and `StickerApp-Self-Contained`) using the `upload-artifact` action.
